O QUE É LÓGICA DE PROGRAMAÇÃO?

A lógica de programação é como uma preparação para a programação, onde o usuário é treinado e preparado para resolver problemas lógicos e matemáticos, enquanto a programação é a aplicação dessa lógica para criar programas através de linguagens próprias.
